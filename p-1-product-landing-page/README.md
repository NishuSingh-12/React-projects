# 👟 Product Landing Page (React)

This is a simple and responsive product landing page built using React. The design is inspired by a Nike-style product page and focuses on clean UI and responsive layout.

## Screenshots

![Desktop UI](public/images/desktop.png)
![Mobile UI](public/images/mobile.png)

## 🚀 Features

- Responsive design (works on mobile and desktop)
- Clean and simple UI
- Reusable components
- Flexbox-based layout
- Image handling using public folder

## 🧩 Components Used

- Navbar
- Hero
- Footer

## 📱 Responsive Design

The layout uses Flexbox and media queries to adjust the UI for different screen sizes.

- Desktop: Text on left, image on right
- Mobile: Image shifts above the text

## 🛠️ Tech Stack

- React
- CSS (Flexbox + Media Queries)
