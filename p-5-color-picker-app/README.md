# 🎨 React Color Picker App

A simple and interactive Color Picker application built using React. This app allows users to select any color using the native color input and instantly apply it as the background of the screen.

## 🚀 Features

- 🎯 Real-time background color change
- 🎨 Uses native HTML color picker
- ⚡ Built with React functional components and hooks
- 🧠 Simple state management using `useState`

## 🛠️ Tech Stack

- React (Functional Components)
- JavaScript (ES6)
- HTML5 (Color Input)
- CSS (Inline Styling)

## 📸 How It Works

- The app uses the `useState` hook to store the selected color.
- When the user picks a color, the `onChange` event updates the state.
- The selected color is applied dynamically to the background of the container.
